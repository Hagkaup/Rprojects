Rprojects - Survival Analyses for time-to-event data

Parametric and semi-parametric survival regression

View on GitHub Download .zip Download .tar.gz

The files needed to carry out all analyses of the example data can be downloaded from this web site.

Files:

Ina.csv is the file read into the R console

Ina.R is the script containing all the codes to carry out the analysis + notes describing the code

Ina.xlsx is the original Excel sheet from which the csv file was created

Survival analysis.pptx is the Power Point presentation presented at the lab meeting
